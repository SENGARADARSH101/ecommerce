# Ecommerce

## Spring Boot Based Project 

## Functionalities of the E-Commerce Application: 

### Admin Side: 

  1. Add Product 

  2. Delete Product 

  3. Add Category 

  4. Delete Category 

  5.Get Orders 

  6.Manage Orders 

  7.Update profiles 

  8.Get Reviews 


### User Side: 

  1.Register  

  2.View Category, Product 

  3.Add Products to cart 

  4.Get Cart Value 

  5.Remove product from cart 

  6.Order Products 

  7.Add reviews and ratings 

  8.Check Order Status
